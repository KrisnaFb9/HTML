Pendahuluan HTML

HTML adalah fondasi dari setiap halaman web. Ketika Anda membuka situs web favorit Anda, apa yang Anda lihat sebenarnya adalah hasil dari kode HTML yang diinterpretasikan oleh browser Anda. HTML digunakan untuk mengorganisir dan merangkai elemen-elemen seperti teks, gambar, tautan, dan formulir di halaman web.

HTML, atau HyperText Markup Language, adalah bahasa dasar yang digunakan untuk membangun halaman web. Ini adalah bahasa yang digunakan untuk membuat struktur dan konten dasar dari halaman web yang Anda lihat di internet.











Struktur Dasar HTML:

Setiap halaman HTML dimulai dengan elemen yang mengapit seluruh konten halaman. Di dalamnya, Anda akan menemukan elemen yang berisi informasi seperti judul halaman dan tautan ke file CSS atau JavaScript. Selanjutnya, elemen mengandung isi sebenarnya dari halaman, seperti teks, gambar, dan tautan.

HTML menggunakan elemen atau "tag" untuk mendefinisikan struktur dan jenis konten di dalam halaman. Misalnya, elemen h1 hingga h6 digunakan untuk judul berjenjang, elemen p untuk paragraf, dan elemen img untuk menampilkan gambar.

HTML memungkinkan pembuatan tautan antara halaman web menggunakan elemen a. Dengan elemen ini, Anda dapat membuat tautan ke halaman lain di dalam situs atau ke situs web lain.











Membuat Daftar:

HTML mendukung pembuatan daftar dengan elemen ul (unordered list) dan ol (ordered list). Di dalamnya, elemen li (list item) digunakan untuk setiap elemen dalam daftar.
